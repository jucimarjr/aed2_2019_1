Para executar o programa:
	Instalar programa WampServer ou XAMPP;
	Executar ambiente Apache;
	Colocar pasta do programa no diretório C:\xampp\htdocs -> Para XAMPP
	Colocar pasta do programa no diretório C:\wamp64\www -> Para WampServer
	Na URL do browser digitar 'http://localhost/doninha'
